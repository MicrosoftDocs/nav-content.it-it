    <span data-ttu-id="c818a-117">Nella pagina vengono elencate tutte le pagine che sono state personalizzate e l'utente a cui appartengono.</span><span class="sxs-lookup"><span data-stu-id="c818a-117">The page lists all the pages that have been personalized and the user it belongs to.</span></span>

    > <span data-ttu-id="c818a-118">Un segno di spunta nella colonna **Personalizzazione legacy** indica che una personalizzazione è stata effettuata solo tramite [!INCLUDE[nav_windows_md](includes/nav_windows_md.md)] e/o è stata effettuata in [!INCLUDE[nav_web_md](includes/nav_web_md.md)] precedentemente a [!INCLUDE[navnow_md](includes/navnow_md.md)].</span><span class="sxs-lookup"><span data-stu-id="c818a-118">A check mark in the **Legacy Personalization** column indicates that the personalization has been done strictly by using [!INCLUDE[nav_windows_md](includes/nav_windows_md.md)] and/or it has been done in [!INCLUDE[nav_web_md](includes/nav_web_md.md)] prior to [!INCLUDE[navnow_md](includes/navnow_md.md)].</span></span> <span data-ttu-id="c818a-119">Gli utenti che provano a personalizzare queste pagine utilizzando [!INCLUDE[nav_web_md](includes/nav_web_md.md)] vengono bloccati a meno che non scelgano di sbloccare la pagina.</span><span class="sxs-lookup"><span data-stu-id="c818a-119">Users who try to personalize these pages by using the [!INCLUDE[nav_web_md](includes/nav_web_md.md)] are locked from doing so unless they choose to unlock the page.</span></span> <span data-ttu-id="c818a-120">Per ulteriori informazioni, vedere [Perché la personalizzazione di una pagina è bloccata](ui-personalization-locked.md). 2. <span data-ttu-id="c818a-121">Selezionare la voce che si desidera eliminare, quindi scegliere l'azione **Elimina**.</span><span class="sxs-lookup"><span data-stu-id="c818a-121">Select the entry that you want to delete, and then choose the **Delete** action.</span></span>

    <span data-ttu-id="c818a-122">L'utente visualizzerà le modifiche dopo l'accesso successivo.</span><span class="sxs-lookup"><span data-stu-id="c818a-122">The user will see the changes the next time they sign-in.</span></span>
