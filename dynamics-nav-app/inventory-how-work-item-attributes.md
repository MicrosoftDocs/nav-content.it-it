# <a name="how-to-work-with-item-attributes"></a><span data-ttu-id="a23de-102">Procedura: Utilizzare gli attributi degli articoli</span><span class="sxs-lookup"><span data-stu-id="a23de-102">How to: Work with Item Attributes</span></span>
<span data-ttu-id="a23de-103">Quando i clienti richiedono informazioni su un articolo, per corrispondenza o in un negozio Web integrato, possono chiedere o cercare in base alle caratteristiche, ad esempio l'altezza e l'anno del modello.</span><span class="sxs-lookup"><span data-stu-id="a23de-103">When customers inquire about an item, either in correspondence or in an integrated web shop, they may ask or search according to characteristics, such as height and model year.</span></span> <span data-ttu-id="a23de-104">Per fornire all'assistenza ai clienti, è possibile assegnare valori di attributo articolo di diversi tipi agli articoli e utilizzarli durante la ricerca degli articoli.</span><span class="sxs-lookup"><span data-stu-id="a23de-104">To provide this customer service, you can assign item attribute values of different types to your items, which can then be used when searching for items.</span></span>

## <a name="to-create-item-attributes"></a><span data-ttu-id="a23de-107">Per creare attributi articolo</span><span class="sxs-lookup"><span data-stu-id="a23de-107">To create item attributes</span></span>
1. <span data-ttu-id="a23de-108">Nell'angolo superiore destro, scegliere l'icona **Cerca pagina o report**, immettere **Attributi articolo**, quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="a23de-108">In the top right corner, choose the **Search for Page or Report** icon, enter **Item Attributes**, and then choose the related link.</span></span>
2. <span data-ttu-id="a23de-109">Nella finestra **Attributi articolo** scegliere l'azione **Nuovo**.</span><span class="sxs-lookup"><span data-stu-id="a23de-109">In the **Item Attributes** window, choose the **New** action.</span></span>
3. <span data-ttu-id="a23de-110">Nella finestra **Attributi articolo** compilare i campi secondo le proprie necessità.</span><span class="sxs-lookup"><span data-stu-id="a23de-110">In the **Item Attribute** window, fill in the fields as necessary.</span></span> <span data-ttu-id="a23de-111">Selezionare il campo per visualizzare una breve descrizione del campo o il collegamento a ulteriori informazioni.</span><span class="sxs-lookup"><span data-stu-id="a23de-111">Choose a field to read a short description of the field or link to more information.</span></span>

<span data-ttu-id="a23de-112">**Nota**: se si seleziona **Opzione** nel campo **Tipo** è possibile scegliere l'azione **Valori attributo articolo** per creare valori per l'attributo dell'articolo.</span><span class="sxs-lookup"><span data-stu-id="a23de-112">**Note**: If you select **Option** in the **Type** field, then you can choose the **Item Attribute Values** action to create values for the item attribute.</span></span> <span data-ttu-id="a23de-113">Per ulteriori informazioni, vedere la sezione "Per creare i valori per gli attributi articolo di tipo Opzione".</span><span class="sxs-lookup"><span data-stu-id="a23de-113">For more information, see the "To create values for item attributes of type Option" section.</span></span>

1. <span data-ttu-id="a23de-115">Nell'angolo superiore destro, scegliere l'icona **Cerca pagina o report**, immettere **Attributi articolo**, quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="a23de-115">In the top right corner, choose the **Search for Page or Report** icon, enter **Item Attributes**, and then choose the related link.</span></span>
2. <span data-ttu-id="a23de-116">Nella finestra **Attributi articolo**, selezionare l'attributo dell'articolo di tipo Opzione per cui si desidera creare i valori quindi selezionare l'azione **Valori attributo articolo**.</span><span class="sxs-lookup"><span data-stu-id="a23de-116">In the **Item Attributes** window, select an item attribute of type Option that you want to create values for, and then choose the **Item Attribute Values** action.</span></span>
3. <span data-ttu-id="a23de-117">Nella finestra **Valori attributo articolo** compilare i campi secondo le proprie necessità.</span><span class="sxs-lookup"><span data-stu-id="a23de-117">In the **Item Attribute Values** window, fill in the fields as necessary.</span></span> <span data-ttu-id="a23de-118">Selezionare il campo per visualizzare una breve descrizione del campo o il collegamento a ulteriori informazioni.</span><span class="sxs-lookup"><span data-stu-id="a23de-118">Choose a field to read a short description of the field or link to more information.</span></span>

## <a name="to-assign-item-attributes-to-items"></a><span data-ttu-id="a23de-119">Per assegnare attributi articolo agli articoli</span><span class="sxs-lookup"><span data-stu-id="a23de-119">To assign item attributes to items</span></span>
1. <span data-ttu-id="a23de-120">Nell'angolo superiore destro, scegliere l'icona **Cerca pagina o report**, immettere **Articoli**, quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="a23de-120">In the top right corner, choose the **Search for Page or Report** icon, enter **Items**, and then choose the related link.</span></span>
2. <span data-ttu-id="a23de-121">Nella finestra **Articoli**, selezionare l'articolo a cui si desidera assegnare gli attributi articoli quindi selezionare l'azione **Attributi**.</span><span class="sxs-lookup"><span data-stu-id="a23de-121">In the **Items** window, select the item that you want to assign item attributes to, and then choose the **Attributes** action.</span></span>
3. <span data-ttu-id="a23de-122">Nella finestra **Valori attributo articolo** scegliere l'azione **Nuovo**.</span><span class="sxs-lookup"><span data-stu-id="a23de-122">In the **Item Attribute Values** window, choose the **New** action.</span></span>
4. <span data-ttu-id="a23de-123">Nel campo **Attributo** scegliere il pulsante AssistEdit e selezionare un attributo articolo esistente.</span><span class="sxs-lookup"><span data-stu-id="a23de-123">Choose the AssistEdit button in the **Attribute** field and select an existing item attribute.</span></span> <span data-ttu-id="a23de-124">In alternativa, scegliere l'azione **Nuovo** per creare prima un nuovo attributo articolo come descritto nella sezione "Per creare attributi articolo".</span><span class="sxs-lookup"><span data-stu-id="a23de-124">Alternatively, choose the **New** action to first create a new item attribute as explained in the "To create item attributes" section.</span></span>
5. <span data-ttu-id="a23de-125">Nel campo **Valore**, immettere il valore dell'attributo articolo, ad esempio "2010" per l'attributo dell'anno del modello.</span><span class="sxs-lookup"><span data-stu-id="a23de-125">In the **Value** field, enter the item attribute value, such as "2010" for the Model Year attribute.</span></span>
6. <span data-ttu-id="a23de-126">Per gli attributi articolo di tipo Opzione, scegliere il pulsante AssistEdit nel campo **Valore** quindi selezionare un valore di attributo articolo.</span><span class="sxs-lookup"><span data-stu-id="a23de-126">For item attributes of type Option, choose the AssistEdit button in the **Value** field and select an item attribute value.</span></span> <span data-ttu-id="a23de-127">In alternativa, scegliere l'azione **Nuovo** per creare prima un nuovo valore di attributo articolo come descritto nella sezione "Per creare i valori per gli attributi articolo di tipo Opzione".</span><span class="sxs-lookup"><span data-stu-id="a23de-127">Alternatively, choose the **New** action to first create a new item attribute value as explained in the "To create values for item attributes of type Option" section.</span></span>
7. <span data-ttu-id="a23de-128">Ripetere i passaggi da 4 a 6 per tutti gli attributi articoli che si desidera assegnare all'articolo.</span><span class="sxs-lookup"><span data-stu-id="a23de-128">Repeat steps 4 through 6 for all item attributes that you want to assign to the item.</span></span>

## <a name="to-assign-item-attributes-to-item-categories"></a><span data-ttu-id="a23de-129">Per assegnare attributi articolo alle categorie articoli</span><span class="sxs-lookup"><span data-stu-id="a23de-129">To assign item attributes to item categories</span></span>
1. <span data-ttu-id="a23de-130">Nell'angolo superiore destro, scegliere l'icona **Cerca pagina o report**, immettere **Categorie articoli**, quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="a23de-130">In the top right corner, choose the **Search for Page or Report** icon, enter **Item Categories**, and then choose the related link.</span></span>
2. <span data-ttu-id="a23de-131">Nella finestra **Categorie articoli**, selezionare la categoria dell'articolo a cui si desidera assegnare gli attributi articolo quindi selezionare l'azione **Modifica**.</span><span class="sxs-lookup"><span data-stu-id="a23de-131">In the **Item Categories** window, select the item category that you want to assign item attributes to, and then choose the **Edit** action.</span></span>
3. <span data-ttu-id="a23de-132">Nella finestra **Scheda categoria articolo**, scegliere l'azione **Nuovo** nella Scheda dettaglio **Attributi**.</span><span class="sxs-lookup"><span data-stu-id="a23de-132">In the **Item Category Card** window, on the **Attributes** FastTab, choose the **New** action.</span></span>
4. <span data-ttu-id="a23de-133">Nel campo **Attributo** scegliere il pulsante AssistEdit e selezionare un attributo articolo esistente.</span><span class="sxs-lookup"><span data-stu-id="a23de-133">Choose the AssistEdit button in the **Attribute** field and select an existing item attribute.</span></span> <span data-ttu-id="a23de-134">In alternativa, scegliere l'azione **Nuovo** per creare prima un nuovo attributo articolo come descritto nella sezione "Per creare un attributo articolo".</span><span class="sxs-lookup"><span data-stu-id="a23de-134">Alternatively, choose the **New** action to first create a new item attribute as explained in the "To create an item attribute" section.</span></span>

6. <span data-ttu-id="a23de-136">Ripetere i passaggi da 4 a 5 per tutti gli attributi articoli che si desidera assegnare alla categoria articolo.</span><span class="sxs-lookup"><span data-stu-id="a23de-136">Repeat steps 4 and 5 for all item attributes that you want to assign to the item category.</span></span>

<span data-ttu-id="a23de-137">**Nota**: gli attributi degli articoli per le categorie articolo padre verranno ereditati dalle categorie articolo figlio.</span><span class="sxs-lookup"><span data-stu-id="a23de-137">**Note**: Item attributes for parent item categories will be inherited to child item categories.</span></span> <span data-ttu-id="a23de-138">Ciò è indicato dal campo **Ereditato da** nella Scheda dettaglio **Attributi**.</span><span class="sxs-lookup"><span data-stu-id="a23de-138">This is indicated by the **Inherited From** field on the **Attributes** FastTab.</span></span> <span data-ttu-id="a23de-139">Per ulteriori informazioni, vedere [Procedura: Classificare gli articoli](inventory-how-categorize-items.md).</span><span class="sxs-lookup"><span data-stu-id="a23de-139">For more information, see [How to: Categorize Item](inventory-how-categorize-items.md).</span></span>

## <a name="to-filter-by-item-attributes"></a><span data-ttu-id="a23de-140">Per filtrare per attributi articolo</span><span class="sxs-lookup"><span data-stu-id="a23de-140">To filter by item attributes</span></span>
1. <span data-ttu-id="a23de-141">Nell'angolo superiore destro, scegliere l'icona **Cerca pagina o report**, immettere **Articoli**, quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="a23de-141">In the top right corner, choose the **Search for Page or Report** icon, enter **Items**, and then choose the related link.</span></span>
2. <span data-ttu-id="a23de-142">Nella finestra **Articoli** scegliere l'azione **Filtra per attributi**.</span><span class="sxs-lookup"><span data-stu-id="a23de-142">In the **Items** window, choose the **Filter by Attributes** action.</span></span>
3. <span data-ttu-id="a23de-143">Nella finestra **Filtra articoli per attributo**, fare clic sul pulsante AssistEdit nel campo **Attributo** e selezionare un attributo dell'articolo.</span><span class="sxs-lookup"><span data-stu-id="a23de-143">In the **Filter Items by Attribute** window, choose the AssistEdit button in the **Attribute** field and select an item attribute.</span></span>
4. <span data-ttu-id="a23de-144">Nel campo **Valore**, fare clic sul pulsante di AssistEdit e selezionare un valore attributo in base a cui filtrare gli articoli.</span><span class="sxs-lookup"><span data-stu-id="a23de-144">In the **Value** field, choose the AssistEdit button and select an attribute value to filter items by.</span></span>

    <span data-ttu-id="a23de-145">**Nota**: è possibile selezionare solo i valori direttamente per gli attributi dell'articolo che hanno valori fissi, ad esempio il colore.</span><span class="sxs-lookup"><span data-stu-id="a23de-145">**Note**: You can only select values directly for item attributes that have fixed values, such as Color.</span></span> <span data-ttu-id="a23de-146">Per gli attributi dell'articolo che hanno valori variabili, quali la larghezza, è necessario specificare il valore dell'attributo dell'articolo selezionando prima una condizione.</span><span class="sxs-lookup"><span data-stu-id="a23de-146">For item attributes that have variable values, such as Width, you must specify the item attribute value by first selecting a condition.</span></span> <span data-ttu-id="a23de-147">Vedere il passaggio 5.</span><span class="sxs-lookup"><span data-stu-id="a23de-147">See step 5.</span></span>
5. <span data-ttu-id="a23de-148">Nel campo **Valore** per un attributo articolo variabile, fare clic sul pulsante di AssistEdit.</span><span class="sxs-lookup"><span data-stu-id="a23de-148">In the **Value** field for a variable item attribute, choose the AssistEdit button.</span></span>
6. <span data-ttu-id="a23de-149">Nella finestra **Specifica valore filtro** nel campo **Condizione**, fare clic sulla freccia rivolta verso il basso e selezionare una condizione.</span><span class="sxs-lookup"><span data-stu-id="a23de-149">In the **Specify Filter Value** window, in the **Condition** field, choose the drop-down arrow and select a condition.</span></span>
7. <span data-ttu-id="a23de-150">Nel campo **Valore**, immettere un valore attributo in base a cui filtrare gli articoli.</span><span class="sxs-lookup"><span data-stu-id="a23de-150">In the **Value** field, enter an attribute value to filter items by.</span></span>

    <span data-ttu-id="a23de-151">**Esempio**: per filtrare gli articoli in cui la descrizione materiale inizia con "blu", compilare i campi come segue: **Attributo** campo: Descrizione materiale, **Condizione** campo: Inizia con **Valore**, campo: blu.</span><span class="sxs-lookup"><span data-stu-id="a23de-151">**Example**: To filter on items where the material description begins with "blue", fill in the fields as follows: **Attribute** field: Material Description, **Condition** field: Begins With, **Value** field: blue.</span></span>
8. <span data-ttu-id="a23de-152">Scegliere il pulsante **OK**.</span><span class="sxs-lookup"><span data-stu-id="a23de-152">Choose the **OK** button.</span></span>
